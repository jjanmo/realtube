# RealTube

- What is RealTube : 리얼클래스(or스쿨라이브) + 유튜브 = 리얼튜브

- 영어공부를 위한 영상 `플랫폼` 사이트를 만들어보자!

<details>
  <summary>Motive</summary>

- 내가 개발 중인 서비스에 대해 기술적으로 좀 더 알고 싶은 마음
- 최초에는 어떤 식으로 개발되었기에 지금과 같은 모습을 지니게 된 것일까에 대한 궁금증
- 내가 생각하는 서비스의 방향성을 적용해보자.
</details>
